@TITLE: Documentation@ @START@

The following summary lists documentations available along the Unix/GNU Windows project. It works under MS-Windows 98, 2000, XP and most certainly will work as is with Vista. At this time, it requires MSYS as the environment to run Unix commands. What this project adds is its own package manager called wpkg. This is a command line only package manager. As it becomes closer to a full version, we will then add an extra layer: apt-get. A famous tool used by debian distributions in order to manage the debian environment.
