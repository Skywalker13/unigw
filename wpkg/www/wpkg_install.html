@TITLE: Installing a package@ @UP: wpkg@ @NEXT: wpkg_remove@ @START@

Installing a UniGW package with wpkg is very simple. The following are the two steps process:

  1. Download the .deb that you want to install
  2. In your MSYS console, type:
    3. 	wpkg --install <package-name>.deb

Once the package was installed, you can delete the .deb file. It it not necessary for the good functioning of the installed package. wpkg keeps the necessary files in its datbase to know how to configure and remove the package as required.

WARNING: wpkg must be installed as /mingw/bin/wpkg.exe for this command to work properly. This is automatic when you install USYS.
@END@
